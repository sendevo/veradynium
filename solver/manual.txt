NAME
   solver - LoRaWAN SOLVER

SYNOPSIS
   solver [OPTIONS]... -f [FILE]

DESCRIPTION:
   This program computes an optimal solution for the VirtualGateway Placement Problem (GPP). 

OPTIONS:
   -h, --help     Display this help message.

EXAMPLES:
   1. solver -f input.json
      - Run the program using the default options.
